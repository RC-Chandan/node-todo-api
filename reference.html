Node JS Todo list api reference

  • Make HTTP.GET request to /api/todos to get list of todos [demo]
  • Make HTTP.GET request to /api/todo/:id to get perticular todo entity
  • Make HTTP.POST to /api/todos to insert todo entity
  • Make HTTP.POST to /api/todos with todo id to update existing todo entity
  • Make HTTP.DELETE request to /api/todos with id as json to delete todo entity
